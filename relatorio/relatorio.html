Relatório

Funcionalidades Implementadas:

  • Sistema de Login
  • Sistema de Logout
  • Listagens globais
  • Listagens específicas
  • Inserir dados
  • Editar dados

Funcionalidades Extras:

  • Editar dados de todas as tabelas

Sistema de Login

Nós utilizamos o modal do bootstrap para criarmos o login do nosso utilizador, para acedermos ao nosso utilizador e a sua respetiva palavra passe utilizamos o mysql.O utilizador só consegue aceder ao resto do nosso site caso esteja com o login efetuado, caso o utilizador tente aceder ao nosso site e não esteja logado, aparece uma mensagem no url a dizer que não tem permissão para aceder o site.

Sistema de Logout

No nosso trabalho é possivel dar logout de duas formas.Uma das formas é com a utilização de cookies, assim após um tempo definido por nós, o utilizador é expulso do site.

A outra forma é terminarmos sessão manualmente, no nosso header(após sessão iniciada) temos a opção Log out, quando primida, o utilizador recebe um pedido de confirmação para sair(modal) e caso confirme,é deslogaddo com sucesso.

Listagens Globais

O nosso header contém a opção table, quando selecionada, aparece uma tabela que contém as cinco opções necessárias(course, faculty, product, student e tent).

Selecionando uma das opções, estabelecemos uma ligação à base de dados de acordo com a opção selecionada, mostrado ao utilizador os respetivos dados da tabela.

Listagens Específicas

No nosso header, quando acedemos à opção table, é possível organizar as tabelas de cada opção por ordem alfabética(nomes) e numérica(preços), também quando selecionamos a opção insert é possível ordenar as tabelas.

Inserir Dados

O nosso header contém a opção insert, quando selecionada, podemos escolher 6 opções de inserção(faculdade, cursos, barracas, alunos, stock e produtos).

Na inserção de produtos e faculdades apenas necessitamos de submeter a informação dos produtos(nome e preço) e da faculdade(nome) respetivamente.Na inserção das restantes opções além da informação acerca da opção é necessário selecionar as opções das drop-down menu,por exemplo,na inserção de um aluno é necessário adicionar o nome e número do aluno,bem como, a faculdade, o curso e a barraca através de 3 drop-down menus.

Editar Dados

Para editarmos os dados da nossa base de dados necessitamos de selecionar a opção insert no nosso header, quando selecionada, podemos escolher uma das 6 opções de inserção, após a escolha basta selecionarmos a opção alterar.

No caso do stock, após submetermos a alteração, aparecerá uma caixa(modal) a questionar se queremos mesmo efetuar esta alteração.

Editar Dados de todas as tabelas

Como implementação extra nós decidimos permitir a edição da faculdade, cursos, barracas, alunos e produtos.

A forma de editar estas opções é igual à forma de editar o stock porém, após a submição não existe uma caixa(modal) a questionar a nossa submissão.
